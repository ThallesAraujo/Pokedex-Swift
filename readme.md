# Pokedex Swift

![Screenshot](/screenshot.png)

Projeto em Swift de pokedex utilizando a [PokeAPI](https://pokeapi.co) 

## Requisitos:

- XCode 12+
- CocoaPods
- iOS 14.1+

## Instalação:

- Executar o comando ``pod install`` na pasta raiz do projeto
- Executar a build no XCode
- e voilà!
